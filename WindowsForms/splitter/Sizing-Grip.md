# Sizing grip

The Sizing Grip appearance is customized by using ShowSizeGrip property in SplitterControl.

{% tabs %}

{% highlight C# %}



this.splitterControl1.ShowSizeGrip = true;

{% endhighlight %}

{% highlight VB %}

Me.splitterControl1.ShowSizeGrip = False

{% endhighlight %}

{% endtabs %}
