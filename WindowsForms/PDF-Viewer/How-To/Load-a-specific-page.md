# Navigating to a specific page in Windows Forms PdfViewerControl

Navigation to a specific page, through code, is possible using GoToPageAtIndex method.

{% tabs %}
{%highlight c#%}

pdfViewerControl1.GoToPageAtIndex(2);

{%endhighlight%}

{%highlight vb%}
