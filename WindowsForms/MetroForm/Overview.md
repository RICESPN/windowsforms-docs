# Overview

[MetroForm](https://help.syncfusion.com/cr/cref_files/windowsforms/tools/Syncfusion.Shared.Base~Syncfusion.Windows.Forms.MetroForm.html) is used to create customizable window for the end userâ€™s applications. The features offered includes resizing, dragging and moving the window. It also supports various built in skins and let the user to control its behavior and appearance .

![](Overview_images/Overview.png)

## Features

* Supports various color schemes customization.
* Help button support.
* Complete support for Caption Bar and Caption Image customization.
* Right to left support.

* **Caption Labels** - Provides support to add Label in desired location on Caption Bar.
