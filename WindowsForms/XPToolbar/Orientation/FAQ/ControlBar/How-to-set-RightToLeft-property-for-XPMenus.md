# How to set RightToLeft property for XPMenus

RightToLeft can be enabled for XPMenus using the below code.

{% highlight c# %}

this.RightToLeft = RightToLeft.Yes;

{% endhighlight %}

{% highlight vbnet %}

Me.RightToLeft = RightToLeft.Yes

{% endhighlight %}
