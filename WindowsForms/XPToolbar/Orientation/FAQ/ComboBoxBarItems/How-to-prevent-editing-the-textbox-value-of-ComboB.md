# How to prevent editing the textbox value of ComboBoxBarItem

To prevent the user from editing the textbox value of ComboBoxBarItem, set Editable property to false.

{% highlight c# %}

comboBoxBarItem1.Editable = false;

{% endhighlight %}

{% highlight vbnet %}

comboBoxBarItem1.Editable = False

{% endhighlight %}
