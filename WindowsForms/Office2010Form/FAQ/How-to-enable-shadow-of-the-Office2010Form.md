# How to enable shadow of the Office2010Form?

Shadow of the Office2010Form can be enabled or disabled using the `DropShadow` property.

{% tabs %}

{% highlight C# %}

this.DropShadow = false ;

{% endhighlight %}

{% highlight VB %}

 Me.DropShadow = false 
 
{% endhighlight %}

{% endtabs %}
