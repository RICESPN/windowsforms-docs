## Get page count

The number of pages in the currently loaded PDF can be found by using PageCount property.


{% tabs %}
{%highlight c#%}

int count = pdfViewerControl1.PageCount;

{%endhighlight%}


{%highlight vb%}
Dim count As Integer = pdfViewerControl1.PageCount

{%endhighlight%}
{% endtabs %}
