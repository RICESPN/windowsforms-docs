# How to enable shadow of the MetroForm?

Shadow of the MetroForm can be enabled or disabled using the `DropShadow` property.

{% tabs %}

{% highlight C# %}

this.DropShadow = false ;

{% endhighlight %}

{% highlight VB %}

 Me.DropShadow = false 
 
{% endhighlight %}

{% endtabs %}
