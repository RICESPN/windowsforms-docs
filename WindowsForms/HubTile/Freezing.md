# Freezing

The Notification and Animation of a HubTile can be frozen.

The following code example shows how to freeze the HubTile.

{% highlight C# %}  

this.HubTile1.IsFrozen = true;

{% endhighlight %}

{% highlight vbnet %} 

Me.HubTile1.IsFrozen = True

{% endhighlight %}
