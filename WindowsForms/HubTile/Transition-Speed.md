# Transition Speed

In HubTile, image transition speed can be increased or decreased by using ImageTransitionSpeed property.

{% highlight C# %} 

this.HubTile1.ImageTransitionSpeed = 3;

 {% endhighlight %}

{% highlight vbnet %} 

Me.HubTile1.ImageTransitionSpeed = 3

{% endhighlight %}
