# Shadow of the Office2007Form

Shadow of the Office2007Form can be enabled or disabled using the `DropShadow` property.

{% tabs %}

{% highlight C# %}

this.DropShadow = false ;

{% endhighlight %}

{% highlight VB %}

 Me.DropShadow = false 
 
{% endhighlight %}

{% endtabs %}
