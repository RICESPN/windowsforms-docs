# Transition Speed

The TransitionSpeed property enables the items in the control to be rotated at a user defined speed. 
{% highlight C# %}


this.carousel1.TransitionSpeed = 2f;
{% endhighlight %}

{% highlight vbnet %}


Me.carousel1.TransitionSpeed = 2f
{% endhighlight %}
