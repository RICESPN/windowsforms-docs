# Applying Themes

The ProgressBarAdv control can now be themed by enabling the ThemesEnabled property.

{% highlight c# %}

this.progressBarAdv1.ThemesEnabled = true;

{% endhighlight %}

{% highlight vbnet %}

Me.progressBarAdv1.ThemesEnabled = True

{% endhighlight %}
