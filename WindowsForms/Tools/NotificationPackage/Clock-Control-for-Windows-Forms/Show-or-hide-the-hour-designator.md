# Show or hide the hour designator

To display or hide the hour designator (AM and PM) in the DigitalClock, the ShowHourDesignator property can be used.

{% highlight c# %}        

          this.clock1.ShowHourDesignator = false;

{% endhighlight %}

{% endhighlight %}
