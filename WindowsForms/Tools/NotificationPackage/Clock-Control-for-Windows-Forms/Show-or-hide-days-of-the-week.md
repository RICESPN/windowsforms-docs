# Show or hide days of the week

To display or hide the week days and current date in the DigitalClock, the DisplayDates property can be used.

 {% highlight c# %}         

          this.clock1.DisplayDates = true;

{% endhighlight %}



{% highlight vbnet %}           

        Me.clock1.DisplayDates = True

{% endhighlight %}
