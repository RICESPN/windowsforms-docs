# Transition Speed

In HubTile, Rotation speed can be increased or decreased by the RotationTransitionSpeed property.

{% highlight C# %}  

this.HubTile1.RotationTransitionSpeed = 3;

{% endhighlight %}



{% highlight vbnet %} 

Me.HubTile1.RotationTransitionSpeed = 3

{% endhighlight %}
