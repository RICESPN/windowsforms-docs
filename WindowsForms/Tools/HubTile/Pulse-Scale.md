# Pulse Scale

In PulsingTile, the zoom level can be set using PulseScale property.

The following code example shows how to set the PulseScale depth.


{% highlight C# %}  

this.HubTile1.PulseScale= 2;

{% endhighlight %}



{% highlight vbnet %} 

Me.HubTile1.PulseScale = 2

{% endhighlight %}
