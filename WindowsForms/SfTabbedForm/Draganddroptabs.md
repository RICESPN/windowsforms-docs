# Drag and drop tabs

SfTabbedForm allow drag and drop the tabs within the form by setting `tabbedFormControl.AllowDraggingTabs` property as true.

{% tabs %}
{% highlight c# %}
tabbedFormControl.AllowDraggingTabs = true;	
{% endhighlight %}
{% highlight vb %}
tabbedFormControl.AllowDraggingTabs = True
{% endhighlight %}
{% endtabs %}
